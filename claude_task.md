# Claude Code 深度交易分析任务

## 任务目标
使用威科夫交易法和四度空间理论，结合期权期货数据分析当前标普500（SPX/SPY）和纳斯达克100（NDX/QQQ）指数走势，给出短期交易计划（1-5天）。

## 使用工具
- GLM-4.7 模型（最大算力）
- 视觉 MCP 服务器（图片分析）
- 搜索 MCP 服务器（获取最新新闻）
- 所有可用的技术分析工具

## 执行步骤

### 第一步：获取最新市场信息
使用搜索MCP工具获取：
1. 标普500最新价格和技术数据
2. 纳斯达克100最新价格和技术数据
3. 最新市场新闻和宏观经济数据
4. 期权数据（PCR、Max Pain、VIX）
5. 期货数据（ES、NQ）

### 第二步：威科夫交易法分析
分析SPX和NDX：
- 识别当前处于哪个阶段（Accumulation/Markup/Distribution/Markdown）
- 寻找关键信号（Spring、Upthrust、SOS、SOW、Test）
- 分析Effort vs Result
- 评估供需关系和成交量

### 第三步：四度空间理论分析
计算SPX和NDX的：
- 价值区间（Value Area High/Low）
- POC（Point of Control）
- TPO分布形态
- 交易日类型

### 第四步：多时间框架技术分析
分析以下时间框架：
- 1小时图：短期趋势和交易机会
- 4小时图：中期趋势确认
- 日线图：主要趋势方向
- 周线图：长期趋势背景

技术指标：
- 趋势指标：MA(20,50,200)、MACD
- 动能指标：RSI(14)、Stochastic、ADX
- 成交量指标：OBV、VWAP

### 第五步：期权和期货数据分析
- Put/Call Ratio（PCR）
- 最大痛苦点（Max Pain）
- VIX 隐含波动率
- IV Skew（Put IV vs Call IV）
- 期权流向（大单分析）
- E-mini 期货升贴水
- 未平仓合约（OI）变化

### 第六步：市场消息和宏观因素分析
- 最新市场新闻
- 美联储政策预期
- 经济数据（CPI、PCE、就业等）
- 地缘政治风险
- 企业盈利季报

### 第七步：生成短期交易计划
对SPX和NDX分别给出：
- 观点：看多/看空/中性
- 入场点位
- 止损位
- 目标位1/2/3
- 风险收益比
- 建议仓位
- 期权策略建议
- 时间框架（1-5天）

### 第八步：风险管理建议
- 主要风险因素
- 对冲策略
- 止损执行纪律
- 仓位管理原则

## 输出要求

### 报告文件
保存为 `CLAUDE_ANALYSIS_REPORT.md`，包含：
1. 完整的分析过程和逻辑
2. 详细的数据和图表
3. 明确的结论和交易建议
4. 风险提示和免责声明

### 报告格式
使用Markdown格式，包含：
- 表格（数据对比）
- 代码块（数据和分析逻辑）
- 清晰的标题结构
- 图表（如果有）

### 报告深度
- 使用最大算力
- 最详细的推理过程
- 尽可能多的token输出
- 充分利用GLM-4.7的能力
