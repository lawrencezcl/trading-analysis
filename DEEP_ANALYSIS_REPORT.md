```

└──────┬──────┘
       ↓
┌─────────────┐
│   Markup    │ (上涨期)
└──────┬──────┘
       ↓
┌─────────────┐
│ Distribution│ (派发期)
└──────┬──────┘
       ↓
┌─────────────┐
│   Markdown  │ (下跌期)
└─────────────┘
```

## 三、四度空间（Market Profile）分析

### 3.1 Market Profile 核心概念

四度空间理论通过分析**时间-价格-成交量-波动率**四维关系来理解市场结构。

**核心指标**:

```
TPO (Time Price Opportunity): 每个价格水平的时间分布
POC (Point of Control): 交易最活跃的价格位
Value Area: 70% TPO集中的价格区间
  - VAH (Value Area High): 价值区间高点
  - VAL (Value Area Low): 价值区间低点
```

**计算方法**:

```python
# 伪代码
prices = [price_list]
tpo_counts = count_tpo_per_price(prices)
total_tpo = sum(tpo_counts)
sorted_prices = sort_by_tpo(tpo_counts)

# 从POC开始扩展到70% TPO
cumulative_tpo = 0
value_area = []
for price in sorted_prices:
    cumulative_tpo += tpo_counts[price]
    value_area.append(price)
    if cumulative_tpo >= total_tpo * 0.7:
        break

VAH = max(value_area)
VAL = min(value_area)
POC = sorted_prices[0]  # TPO最多的价格
```

## 九、风险管理

### 9.1 主要风险因素

| 风险类型 | 具体风险 | 影响程度 | 应对措施 |
|---------|---------|---------|---------|
| **政策风险** | 美联储意外鹰派 | 高 | 降低仓位，使用期权对冲 |
| **经济数据** | CPI/PCE超预期 | 中 | 关注数据发布，调整仓位 |
| **地缘政治** | 中东/俄乌冲突升级 | 中-高 | 保持流动性，分批操作 |
| **盈利季报** | 科技巨头不及预期 | 中 | 避免集中单一标的 |
| **技术性** | 突破失败，假突破 | 中 | 严格止损，等待确认 |

**风险评估**:
```
综合风险等级: 中等 (6/10)
可控性: 高
建议: 控制仓位，分散风险
```

### 9.2 对冲策略

**期权对冲**:

| 策略 | 适用场景 | 成本 | 效果 |
|------|---------|------|------|
| **保护性Put** | 持有现货，对冲下跌 | 中等 | 完全对冲下跌 |
| **领口策略** | 持有现货，限制损失和收益 | 低 | 部分对冲 |
| **跨式组合** | 预期大波动，不确定方向 | 高 | 享受波动 |

**推荐对冲方案**:
```
1. 主要持仓: 买入Call或持有现货
2. 对冲: 买入OTM Put，Delta -0.15 到 -0.20
3. 成本: 期权权利金约占市值的 1-2%
4. 效果: 最大损失限制在 3-4%
```

### 9.3 止损执行纪律

**止损原则**:

| 原则 | 说明 |
|------|------|
| **开仓前设定** | 开仓前必须设定止损位 |
| **严格执行** | 止损位一旦触及，立即执行 |
| **不移动止损** | 不允许随意移动止损位（保护性移动除外） |
| **比例限制** | 单笔交易风险不超过总资金的2% |
| **日内复盘** | 每日收盘前检查止损位 |

**止损位设定方法**:
```
技术止损:
- 关键支撑位下方
- ATR倍数 (1.5-2倍)
- 前期低点下方

金额止损:
- 固定金额: 如1000美元
- 固定比例: 如2%

时间止损:
- 持有时间超过5天未达预期
- 考虑平仓，重新评估
```

### 9.4 仓位管理原则

**仓位分配**:

| 仓位类型 | 比例 | 说明 |
|---------|------|------|
| **单笔交易** | 2-3% | 控制单笔风险 |
| **同一品种** | ≤10% | 避免集中风险 |
| **总仓位** | 50-70% | 保持流动性 |
| **现金储备** | 30-50% | 应对市场波动 |

**杠杆控制**:
```
总杠杆: 不超过3倍
- 单一品种: ≤2倍
- 组合总杠杆: ≤3倍
- 杠杆计算: (Delta加权持仓) / 净值
```

**分散投资**:
```
1. 指数分散: SPX, NDX, 其他指数
2. 板块分散: 科技, 金融, 消费等
3. 时间分散: 不同到期日的期权
4. 策略分散: 多头, 对冲, 套利
```

**加仓规则**:
```
允许加仓的条件:
1. 当前仓位盈利 ≥5%
2. 趋势确认，技术指标支持
3. 风险收益比良好
4. 总仓位不超过限制

禁止加仓的情况:
1. 当前仓位亏损
2. 追涨杀跌
3. 情绪化操作
4. 超过风险限额
```
